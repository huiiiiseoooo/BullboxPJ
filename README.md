# BullboxPJ
순수 java를 이용해서 FTP 서버 구현하기
 --------
## 기능구현

### 로그인
- [ ] USER 사용자이름 서버에 보냄
- [ ] PASS 비밀번호 입력(사용자 이름 선행)

### 사용자 기본 기능
- [ ] LIST 서버 파일 목록 표시
- [ ] STOR 서버에 파일 업로드
- [ ] DOWN 서버에서 파일 다운로드
- [ ] DELE 서버 파일 삭제 요청
  - [ ] (본인이 업로드한 파일만 삭제 가능)
- [ ] QUIT 서버 연결 종료
- [ ] PWD 현재 서버 디렉토리 확인
- [ ] CWD 현재 작업중인 디렉토리 변경
